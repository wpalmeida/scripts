# Linux Basic Commands

## Folders and files manipulation
Create a folder
```
mkdir <folder_name>
```
Create a file
```
touch <file_name>
```
Edit a file
```
nano <file_name>
vi <file_name>
vim <file_name>
```
Search terms in a file or command
```
<file_or_command> | grep <term>
```
Change permission to make a file executable
```
chmod +x <file_name>
```
